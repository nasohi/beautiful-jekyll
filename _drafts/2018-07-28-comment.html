--- layout: post title: Cara memasang ZSH di GNU/Linux (Debian) date: '2018-07-28T21:25:00.002+07:00' author: Nasohi Ciptandani tags: modified_time: '2018-07-28T21:25:26.296+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-551456842103659735 blogger_orig_url: https://www.blogger.com/comment.g?blogID=7821506235811301823&postID=551456842103659735 --- 1. apt-get update
2. apt-get install wget git zsh

Mengganti shell root ke /bin/zsh
3. chsh -s /bin/zsh root 

Untuk user baru
4. useradd -d /home/nasohi -s /bin/zsh nasohi
5. passwd nasohi


Install Oh-My-Zsh framework
6. wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh
