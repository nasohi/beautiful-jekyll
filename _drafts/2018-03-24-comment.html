--- layout: post title: Install NodeJS di Debian/Ubuntu date: '2018-03-24T20:25:00.003+07:00' author: Nasohi Ciptandani tags: - Linux - NodeJS modified_time: '2018-03-24T20:51:31.419+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-6088978018773489428 blogger_orig_url: https://www.blogger.com/comment.g?blogID=7821506235811301823&postID=6088978018773489428 --- Mendownload repository untuk mendapatkan software NodeJS

$ curl -sL https://deb.nodesource.com/setup_6.x
$ chmod + x setup_6.x
# ./setup_6.x

Instal NodeJS

Jadi, sekarang setelah Anda mengatur repositori, Anda dapat menginstal NodeJS secara normal melalui `apt`.

# apt get install nodejs
