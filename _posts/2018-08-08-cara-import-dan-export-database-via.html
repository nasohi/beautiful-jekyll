--- layout: post title: Cara import dan export database via command line di MySQL date: '2018-08-08T15:22:00.006+07:00' author: Nasohi Ciptandani tags: - MySQL modified_time: '2018-08-08T15:22:51.159+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-2155952593482195103 blogger_orig_url: https://nciptandani.blogspot.com/2018/08/cara-import-dan-export-database-via.html --- Cara export:

mysqldump -u username -p database_name > data-dump.sql

Cara import:

mysql -u root -p CREATE DATABASE new_database; mysql -u username -p new_database < data-dump.sql
