Di dalam pengembangan sebuah produk website biasanya programmer menggunakan cara native atau framework. Cara native adalah programmer membuat code sendiri dari source code php, sedangkan cara framework adalah programmer menggunakan code yang sudah ada sebelumnya yang dibuat oleh programmer lain atau programmer itu sendiri.  Seperti yang kita tahu Laravel adalah PHP framework, framework menurut saya adalah sebuah kerangka kerja untuk membuat sebuah produk, dengan framework tentunya akan membuat pekerjaan kita menjadi lebih cepat.

Pada kali ini akan membahas langkah-langkah installasi laravel


1. Yang pertama pastikan sudah terinstall XAMPP atau LAMP dan service nya berjalan
2. Yang kedua install composer, software nya bisa di download di https://getcomposer.org/download/
3. Yang ketiga download laravel, masuk ke cmd, pindah ke directory htdocs dan ketikkan  composer create-project laravel/laravel laravelapp --prefer-dist




Tunggu download file laravel sampai selesai, file-file laravel akan terletak di xampp\htdocs\laravelapp

pindah ke direktori laravelapp , ketikkan php artisan serve  agar  aplikasi laravel berjalan.

Cobalah dengan masuk ke browser dan akses localhost:8000 atau localhost\laravelapp\public
