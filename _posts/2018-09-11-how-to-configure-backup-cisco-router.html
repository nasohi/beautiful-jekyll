Configure configuration backup on BR3 router:
a. Backup copy of running configuration should be automatically saved on BACKUP
server using TFTP each time configuration is saved (copied to startup);
b. Use following naming convention for backup files: <hostname>-<time>.cfg
Location for configuration backup files is /srv/tftp/ on BACKUP server
