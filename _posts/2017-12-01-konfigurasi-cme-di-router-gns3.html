Hari ini saya akan memberikan langkah-langah CME bekerja pada router GNS3 / Dynamips.
Langkah yang saya ikuti:

- Buat Proyek GNS3 baru

- Dapatkan versi IOS "c3745-adventerprisek9-mz.124-9.bin". Pastikan gunakan router versi itu, coba jalankan dulu dan ketikkan show telephony-service, pastikan command tersedia

- Dapatkan file CME "cme-full-4.3.0.0.tar"

- Buat router baru 3745

- Edit properties dan ganti ruang PCMCI disk0 menjadi misal  100MB itu sudah cukup

- Jalankan Router

- Siapkan server TFTP di mana Anda akan meletakkan file "cme-full-4.3.0.0.tar" di router 3745

- Pastikan Anda bisa melakukan ping ke server TFTP dari router (sehingga server TFTP dapat diakses dari router)

- Lakukan "erase flash:" di router

- Lakukan "format flash:" di router untuk membuat filesystem DOS

- Lakukan "  archieve tar / xtract tftp: //x.x.x.x/cme-full-4.3.0.0.tar flash:  "  (X = alamat IP server TFTP )

- Pastikan semua file terekstrak  ke dalam flash router.
