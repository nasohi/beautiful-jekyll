Task:

LOGIN BANNER
Must be shown before the login prompt. Must appear for local and network logins.
“Welcome to [hostname] – LKSN 2018”

How to:

For local login edit file /etc/issue

For network login (ssh) edit script Banner in /etc/ssh/sshd_config
Banner /etc/issue

Note:
Edit /etc/motd for banner after login

Video:





Untuk mengganti hostname dengan script bisa menggunakan cara berikut
ganti hostname secara dynamic sesuai dengan hostname server dengan script  `hostname` jangan lupa tanda ` ` pada hostname.

Untuk script change_banner.sh dengan isi seperti berikut misalnya

echo "*** Welcome in `hostname` *** " > /etc/issue

cp /etc/issue /etc/issue.net

Install sshpass

sshpass -p 'Skill39' ssh root@10.1.1.10 'bash -s' < change_banner.sh
