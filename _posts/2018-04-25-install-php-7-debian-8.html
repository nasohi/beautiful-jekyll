Menambahkan Repositori ini ke sistem Anda
Anda dapat memperbarui sistem Anda dengan paket dari PPA ini dengan menambahkan ppa: ondrej / php ke Sumber Perangkat Lunak sistem Anda.

apt-get install apt-transport-https lsb-release ca-certificateswget -O /etc/apt/trusted.gpg.d/php.gpg https://packages.sury.org/php/apt.gpgecho "deb https://packages.sury.org/php/ $ (lsb_release -sc) main"> /etc/apt/sources.list.d/php.list

Memasang PHP
Sekarang karena kami memiliki PPA ondrej yang tersedia sebagai sumber, kita dapat melanjutkan dan menginstal PHP seperti biasa dengan apt-get, kita hanya perlu menentukan versi yang ingin kita gunakan.

 apt-get install php7.1
Memasang Beberapa Paket Tambahan
apt-get install php7.1-cli php7.1-common php7.1-curl php7.1-gd php7.1-json php7.1-mbstring php7.1-mysql php7.1-opcache php7.1-readline php7.1-xml
Verifikasi Pasang
Untuk memverifikasi PHP7.1 diinstal, jalankan saja yang berikut untuk melihat versi yang Anda gunakan.

php -v
