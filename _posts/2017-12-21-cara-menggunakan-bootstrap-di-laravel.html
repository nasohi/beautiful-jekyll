--- layout: post title: Cara menggunakan bootstrap di laravel date: '2017-12-21T00:49:00.003+07:00' author: Nasohi Ciptandani tags: - Laravel - Bootstrap modified_time: '2017-12-21T00:49:38.788+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-8363352711338251393 blogger_orig_url: https://nciptandani.blogspot.com/2017/12/cara-menggunakan-bootstrap-di-laravel.html --- Secara default bootstrap sudah bisa di gunakan namun file bootstrap masih mengambil dari internet (cdn), berikut adalah cara agar file boostrap offline, pertama download dulu file bootstap nya.
Kemudian buka file app.blade.php yang terletak di /resources/views/layout, hapus jalur CDN Anda

Ekstrak paket bootstrap dan kemudian

salin bootstrap.min.css ke folder publik / css Anda
salin bootstrap.min.js ke folder publik / js Anda
Tambahkan yang berikut ke area kepala app.blade.php

<link href = "{{ "{{" }}asset ('css / bootstrap.min.css')}}" rel = "stylesheet">

Sebelum penutup </ body> di app.blade.php

     <! - jQuery (diperlukan untuk plugin JavaScript Bootstrap) ->
     <script src = "https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"> </ script>
     <! - Sertakan semua plugin yang dikompilasi (di bawah), atau sertakan file individual sesuai kebutuhan ->
     <script src = "{{ "{{" }}asset ('js / bootstrap.min.js')}}"> </ script>

Anda mungkin perlu melakukan hal yang sama dengan jQuery jika Anda memerlukannya tanpa berasal dari CDN
