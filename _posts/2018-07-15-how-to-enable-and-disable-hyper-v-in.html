OPSI 1 - UNTUK MENGAKTIFKAN DAN MENONAKTIFKAN HYPER-V DI WINDOWS

1. Buka Control Panel\Programs\Programs and Features\Turn Windows features on or off

2. Check untuk mengaktifkan Hyper-V atau Uncheck untuk menonaktifkan Hyper-V

3. Reboot

OPSI 2 -  POWERSHELL

1. Buka PowerShell

2. Untuk Mengaktifkan Hyper-V

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V â€“All

3. Untuk Menonaktifkan Hyper-V

Disable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V-All

4. Ketik Y dan tekan Enter ketika diminta untuk me-restart komputer.

OPSI 3 - MENGGUNAKAN DISM COMMAND

1. Buka CMD

2. Untuk Mengaktifkan Hyper-V

dism.exe /Online /Enable-Feature:Microsoft-Hyper-V /All

3. Untuk Menonaktifkan Hyper-V

dism.exe /Online /Disable-Feature:Microsoft-Hyper-V-All

4. Ketik Y dan tekan Enter ketika diminta untuk me-restart komputer.
