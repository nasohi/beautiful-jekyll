Pembukaan dulu ya, seperti yang kita tahu, GNS3 adalah software simulasi jaringan yang terkenal bisa mensimulasikan ios cisco, iou cisco, virtual machine, dan banyak fitur lainnya.

Kali ini saya mau berbagi pengalaman ketikan menginstall GNS3 di Linux, distro yang saya pakai adalah Ubuntu Linux. Berikut adalah langkah-langkahnya:

Distribusi berbasis Ubuntu (hanya 64-bit)

Petunjuk ini ditujukan untuk Ubuntu dan semua distro berbasis Linux Ubuntu

sudo add-apt-repository ppa:gns3/ppa 
sudo apt-get update 
sudo apt-get install gns3-gui

Jika kita ingin menjalan IOU juga

sudo dpkg --add-architecture i386 
sudo apt-get update  
sudo apt-get install gns3-iou

Semoga bermanfaat teman-teman
