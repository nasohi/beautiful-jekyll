--- layout: post title: Cara zip/unzip di linux command line date: '2018-09-03T17:41:00.004+07:00' author: Nasohi Ciptandani tags: - Command modified_time: '2018-09-03T17:41:48.378+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-7870377019001741290 blogger_orig_url: https://nciptandani.blogspot.com/2018/09/cara-zipunzip-di-linux-command-line.html --- Cukup mudah untuk zip di linux command line, kali ini menggunakan perintah zip

#apt-get install zip

caranya

buatlah sebuah folder dan masukkan file-file kedalam folder tersebut untuk dizip

misalnya ada file1.txt file2.jpg file3.doc
buatlah folder dokumen,

#mkdir dokumen
#cp file1.txt file2.jpg file3.doc dokumen atau #cp file* dokumen

Selanjutnya tinggal gunakan perintah zip, dengan format zip -r nama_dokumen.zip folder_dokumen
option -r artinya recursive

#zip -r dokumen.zip dokumen

Untuk unzip gunakan perintah

#unzip dokumen.zip
