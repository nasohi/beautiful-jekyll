--- layout: post title: Menggunakan perintah Find untuk mencari file di Linux date: '2018-08-02T16:14:00.003+07:00' author: Nasohi Ciptandani tags: - Linux - Command modified_time: '2018-08-02T16:14:27.263+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-804942412710399970 blogger_orig_url: https://nciptandani.blogspot.com/2018/08/menggunakan-perintah-find-untuk-mencari.html --- Berikut adalah perintah nya:

root@nas-srv1:~# find / -name 'Laporan*'

Perintah diatas mencari file dengan nama Laporan*, tanda asterisk * artinya semua nama akhiran dari nama Laporan. Bisa juga ditaruh diawal.  '*Laporan*' artinya semua nama file yang dicari yang mengandung kata 'Laporan'.

Untuk fitur lebih lanjut
#man find
