--- layout: post title: Increase Bash History Size date: '2018-12-10T11:31:00.004+07:00' author: Nasohi Ciptandani tags: - Bash modified_time: '2018-12-10T11:31:50.982+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-6708733719655863227 blogger_orig_url: https://nciptandani.blogspot.com/2018/12/increase-bash-history-size.html --- Increase HISTSIZE – the number of commands to remember in the command history (the default value is 500).

#export HISTSIZE=10000

Increase HISTFILESIZE – the maximum number of lines contained in the history file (the default value is 500).

#export HISTFILESIZE=10000

#source ~/.bashrc
