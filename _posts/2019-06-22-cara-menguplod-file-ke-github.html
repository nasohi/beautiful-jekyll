--- layout: post title: Cara menguplod file ke github date: '2019-06-22T18:37:00.002+07:00' author: Nasohi Ciptandani tags: - Git modified_time: '2019-06-22T18:37:21.487+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-4891839527654799377 blogger_orig_url: https://nciptandani.blogspot.com/2019/06/cara-menguplod-file-ke-github.html --- Buat account github
Buat repository

Gunakan alamat repository yang akan dipakai, contoh: 

git remote add origin https://github.com/username/new_repo

Ke direktori file file yang akan di uplod

Misalnya

cd /home/myproject
git init
git add nama_file atau nama_direktori atau . (tanda titik untuk semua file)
git commit -m "My first project"  (pesan untuk catatan)
git push origin master

Cek

git status
