https://github.com/PowerShell/PowerShell/releases/tag/v6.0.3

An then execute the following in the terminal:

wget https://github.com/PowerShell/PowerShell/releases/download/v6.0.3/powershell_6.0.3-1.debian.9_amd64.deb

dpkg -i powershell_6.0.3-1.debian.9_amd64.deb

apt-get install -f

Run PowerShell
