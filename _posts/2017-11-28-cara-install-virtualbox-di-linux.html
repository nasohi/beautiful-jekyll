--- layout: post title: Cara install VirtualBox di Linux date: '2017-11-28T15:27:00.000+07:00' author: Nasohi Ciptandani tags: - VirtualBox - DKMS modified_time: '2017-11-28T15:27:16.816+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-4209781232876653413 blogger_orig_url: https://nciptandani.blogspot.com/2017/11/cara-install-virtualbox-di-linux.html --- VirtualBox adalah salah satu software virtualisasi buatan oracle. Dengan VirtualBox kita bisa menjalankan sistem operasi untuk keperluan banyak hal seperti untuk software testing, network simulation, dll. 

Berikut adalah langkah-langkah untuk menginstall VirtualBox di Linux, kali ini saya menggunakan Kali Linux.

Pastikan Kali Linux sudah mempunyai repository, kemudian update untuk mendapatkan paket-paket software terbaru

#pico /etc/apt/sources.list

deb http://http.kali.org/kali kali-rolling main contrib non-free


#apt-get update

Setelah update Anda perlu menginstal paket dkms.

#apt-get install dkms

Dynamic Kernel Module Support (DKMS) adalah program / framework untuk pembuatan modul kernel Linux. https://github.com/dell/dkms


Kemudian tinggal install VirtualBox nya dengan perintah dibawah ini

#apt-get install virtualbox

Setelah selesai bisa lihat aplikasi virtualbox di menu apps atau jalankan via terminal ketikkan #virtualbox

Sekian tutorial kali ini, semoga bermanfaat teman teman.
