Aktifkan modul rewrite dengan cara:

a2enmod rewrite

Buka file konfigurasi website anda misal di /etc/apache2/sites-available/000-default.conf dan kemudian cari bagian:

<Directory /var/www/>
        Options Indexes FollowSymLinks
        AllowOverride None
        Require all granted
</Directory>

kemudian ubah menjadi seperti dibawah ini:

<Directory /var/www/>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
</Directory>
