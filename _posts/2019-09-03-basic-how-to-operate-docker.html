Seperti yang kita tahu bahwa container lebih ringan daripada virtual machine karena container menggunakan shared kernel dari host, namun untuk operasional sama seperti virtual machine yang bisa di create, start, stop, run, dll. Berikut adalah perintah-perintah dasar container menggunakan docker:

Example:

Download centos image
Image ini yang akan digunakan sebagai blueprint dari container

docker pull centos

Melihat image yang sudah di download

docker images

Membuat/menjalankan container

docker run -t -d --hostname=centos7-srv1 --name centos7-srv1 centos

Masuk ke terminal container

docker exec -it centos7-srv1 bash

Melihat proses container

docker ps -a

Start docker

docker start container_name

Stop container

docker stop container_name

Delete container

docker rm container_name

Help

man docker
