--- layout: post title: Cara menginstall Packet Tracer 7 di Ubuntu date: '2018-01-14T15:06:00.001+07:00' author: Nasohi Ciptandani tags: - Packet Tracer modified_time: '2018-01-14T15:06:35.641+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-8087032188873261650 blogger_orig_url: https://nciptandani.blogspot.com/2018/01/cara-menginstall-packet-tracer-7-di.html --- Packet Tracer adalah software simulasi jaringan yang dibuat oleh Cisco dengan tujuan untuk mempelajari perangkat jaringan dan edukasi untuk mendapatkan sertifikasi CCNA, dll.

Buka terminal. Kemudian gunakan perintah berikut:

cd Downloads

Anda akan melihat file yang tertera di folder Downloads. Di komputer saya nama file adalah PacketTracer7.tar.gz

tar -zxvf PacketTracer7.tar.gz

Sekarang beberapa output akan muncul, lalu ubah file directory (cd) sebagai berikut:

cd PacketTracer70 /

./install
