Soal: Login sebagai root, buatlah database dengan nama skill39 kemudian buatlah sebuah user mysql dengan nama: dbadm1, password: admp4sswd dan berikan akses hanya ke database skill39, user tersebut hanya bisa mengakses dari localhost, gunakan ssh tunnel jika ingin mengakses database secara remote.

Jawab:

mysql -u root -p

mysql> create database skill39;

mysql> CREATE USER 'dbadm1'@'localhost' IDENTIFIED BY 'admp4sswd';

mysql> GRANT ALL PRIVILEGES ON skill39.* TO 'dbadm1'@'localhost';

mysql> flush privileges;
