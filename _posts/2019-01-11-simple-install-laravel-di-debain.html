Berikut rangkuman step by step nya, buat sekedar catatan, untuk lebih lengkapnya bisa di crosscheck dengan website/blog tutorial lain

sudo apt-get update 
sudo apt-get upgrade
sudo apt-get install curl

Install PHP dan kebutuhan PHP modules untuk Laravel

sudo apt-get install php-cli php-gd php-mbstring php-mcrypt php-zip php-opcache php-xml

Install Composer
Composer adalah dependency manager untuk PHP

sudo curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer

Install Laravel

sudo composer create-project --prefer-dist laravel/laravel my_app
cd my_app
php artisan serve

Laravel development server started: <http://127.0.0.1:8000>

Untuk listen di IP lain bisa menggunakan perintah berikut:

php artisan serve --host=192.168.100.100

Kalau sudah muncul seperti diatas Laravel sudah bisa digunakan dengan konfigurasi diatas,  agar laravel bisa diakses melalui Domain,IP,HTTP/HTTPS maka perlu menginstall Apache/Nginx.
