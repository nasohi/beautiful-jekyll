Soal: SSH Server

 Install and configure SSH Server
- Use key based for SSH authentication
- Make sure user “batik” in LNXCLT-INT and LNXCLT-EXT can SSH to user
root @LNXSRV-A without password
- Allow root access
- Change SSH port default to 2018
