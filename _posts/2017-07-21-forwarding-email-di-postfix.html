Forwarding emails dapat dikonfigurasi melalui file aliases di /etc/aliases
jalankan perintah dibawah ini untuk memasukkan konfigurasi kedalam postfix.conf

postconf -e "alias_maps = hash:/etc/aliases"

Isi file /etc/aliases seperti contoh dibawah ini:

root: nasohi

atau

nasohi: myemail@example.com

atau juga


nasohi: nasohi myemail@example.com

Jalankan perintah dibawah agar tertulis didalam databases aliases

newaliases

Reload postfix

/etc/init.d/postfix reload

Referensi : https://wiki.debian.org/Postfix
