--- layout: post title: Cara install laravel di linux date: '2018-01-12T22:33:00.001+07:00' author: Nasohi Ciptandani tags: - Laravel modified_time: '2018-01-12T22:37:49.315+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-2850446560331976982 blogger_orig_url: https://nciptandani.blogspot.com/2018/01/cara-install-laravel-di-linux.html --- Berikut adalah langkah-langkah ringkas untuk install Laravel di GNU/Linux, saat itu saya menggunakan distro Debian. Seharusnya langkah-langkah nya masih mirip mirip jika anda menggunakan distro seperti Ubuntu, dll.

Step 1 - Install LAMP

Install PHP 7

apt-get install python-software-properties
add-apt-repository ppa:ondrej/php
apt-get update
apt-get install -y php7.1 php7.1-mcrypt php7.1-gd php7.1-mbstring php7.1-xml

Install Apache2

apt-get install apache2 libapache2-mod-php7.1

Install MySQL

apt-get install mysql-server php7.1-mysql

Step 2 – Install Composer



curl -sS https://getcomposer.org/installer | php
mv composer.phar /usr/local/bin/composer
chmod +x /usr/local/bin/composer

Step 3 – Install Laravel

cd /var/www
git clone https://github.com/laravel/laravel.git
cd /var/www/laravel
composer install

chown -R www-data.www-data /var/www/laravel
chmod -R 755 /var/www/laravel
chmod -R 777 /var/www/laravel/storage

Step 4 – Set Encryption Key


mv .env.example .env

php artisan key:generate

Application key [base64:QUiKzqa2tFhy56874sP0Ln7ohJmP3BPlyMhmz7lQBwA=] set successfully.

Sekarang edit config/app.php configuration file di dalam /var/www/laravel/config/app.php

#Gunakan key diatas untuk edit bagian dibawah ini / bagian script 'key'

 'key' => env('APP_KEY', 'base64:QUiKzqa2tFhy56874sP0Ln7ohJmP3BPlyMhmz7lQBwA='),
 'cipher' => 'AES-256-CBC',

Simpan hasil edit file app.php

Sekarang laravel sudah berhasil terinstall, jalankan perintah  php artisan serve untuk menjalankan laravel nya.

Sekian tutorial ringkas cara install Laravel di GNU/Linux. Semoga menjadi catatan yang bermanfaat.
