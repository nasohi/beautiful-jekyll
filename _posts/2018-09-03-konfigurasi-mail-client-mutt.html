--- layout: post title: Konfigurasi mail client Mutt date: '2018-09-03T13:50:00.004+07:00' author: Nasohi Ciptandani tags: - Postfix - Mail modified_time: '2018-09-03T13:50:59.733+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-3485530686185264749 blogger_orig_url: https://nciptandani.blogspot.com/2018/09/konfigurasi-mail-client-mutt.html --- Mutt adalah mail client dengan tampilan CLI

#apt-get install mutt

Login user, buatlah file konfigurasi baru untuk mutt di ~/.muttrc dengan script berikut:

set realname = "Nasohi Ciptandani"
set from = "nasohi@skill39.id"
set use_from = yes
set envelope_from = yes
set smtp_url = "smtp://nasohi@skill39.id@smtp.skill39.id:25/"
set smtp_pass = "p4ssw0rd"

Simpan file konfigurasi diatas

Berikut mencoba mengirim email melalui command line

#mutt -s "Test from mutt" unyil@skill39.id < /tmp/message.txt
