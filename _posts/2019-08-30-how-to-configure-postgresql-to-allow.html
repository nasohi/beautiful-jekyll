--- layout: post title: How to Configure PostgreSQL to allow remote connection date: '2019-08-30T11:02:00.001+07:00' author: Nasohi Ciptandani tags: - PostgreSQL modified_time: '2019-08-30T11:02:07.982+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-3169987859135709340 blogger_orig_url: https://nciptandani.blogspot.com/2019/08/how-to-configure-postgresql-to-allow.html --- Example:
db-srv-10 with ip 192.168.1.1
user database: default postgres

Configure:

find / -name "postgresql.conf"

/var/lib/pgsql/data/postgresql.conf

vi /var/lib/pgsql/data/postgresql.conf

listen_addresses = 'localhost'

with

listen_addresses = '*'

find / -name "pg_hba.conf"

vi /var/lib/pgsql/data/pg_hba.conf

host    all             all              192.168.1.024                       md5

Test:

 psql -h 192.168.1.10 -U postgres
