Supaya database server bisa diremote dari pc lain, beri tanda pagar pada script:

#skip-networking
dan edit
bind-address=YOUR-SERVER-IP

Buatlah sebuah database itnsa untuk competitior1 dengan password: Skill39 diakses dari IP 1.2.3.4:
Jawab:
mysql> CREATE DATABASE itnsa;
mysql> GRANT ALL ON itnsa.* TO competitor1@'1.2.3.4' IDENTIFIED BY 'Skill39';

Tambahan
Cara edit

mysql> use mysql;
mysql> update db set Host='1.2.3.5' where Db='itnsa';
mysql> update user set Host='1.2.3.5' where user='competitor1';
mysql> update user set password=PASSWORD("NEWPASSWORD") where user='it1';

mysql> flush privileges;

Cara akses

#mysql -u root -h YOUR-SERVER-IP -p
