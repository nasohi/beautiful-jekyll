--- layout: post title: Enable remote management windows server core date: '2018-09-01T15:47:00.000+07:00' author: Nasohi Ciptandani tags: - ITNSA - Windows modified_time: '2018-09-01T16:05:03.934+07:00' blogger_id: tag:blogger.com,1999:blog-7821506235811301823.post-3109537504090123318 blogger_orig_url: https://nciptandani.blogspot.com/2018/09/enable-firewall-management-windows.html --- Berikut perintah nya

netsh advfirewall set currentprofile settings remotemanagement enable

Jalankan perintah diatas di windows server core, kemudian lakukan di windows GUI untuk meremote windows firewall pada server core. Dengan cara buka MMC, add snap-in windows firewall pilih another computer, masukkan nama/ip windows server core.

Untuk rule remote disk management gunakan perintah berikut, jalankan perintah berikut di windows server core dan GUI.

netsh advfirewall firewall set rule group=”Remote Volume Management” new enable=yes

Dan banyak hal-hal lainnya yang bisa diremote pada server core dari server GUI.
